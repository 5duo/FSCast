# 爱奇艺DLNA投屏签名问题说明

## 问题描述

当使用爱奇艺APP进行DLNA投屏时，视频无法播放，出现"Source error"错误。

## 根本原因

爱奇艺采用了**多层DRM（数字版权管理）保护机制**：

### 1. **URL签名机制**
- 每个视频URL都包含动态生成的签名参数
- 签名包含：时间戳、设备ID、用户ID、视频ID等
- 签名有时效性（通常几分钟）

### 2. **签名验证流程**
```
手机APP获取播放URL
    ↓
生成签名（包含设备指纹、时间戳等）
    ↓
发送到车机DLNA接收器
    ↓
车机请求视频URL
    ↓
爱奇艺服务器验证签名
    ↓
签名无效 → 返回错误 A00001
```

### 3. **关键参数**
从URL中可以看到这些签名参数：
```
tm: 时间戳
tvid: 视频ID
k_uid: 用户ID  
sgti: 会话ID
vf: 验证码
sign: 签名
px: 加密参数
```

## 为什么车机无法播放？

1. **设备ID不匹配**
   - 手机生成的URL绑定到手机的设备ID
   - 车机的设备ID不同，导致验证失败

2. **时间戳过期**
   - URL从手机发送到车机有延迟
   - 签名中的时间戳可能已过期

3. **用户认证**
   - URL包含用户特定的认证信息
   - 车机无法使用该用户的认证

4. **复杂的加密算法**
   - 爱奇艺使用专有的加密算法
   - 需要逆向工程才能实现

## 当前实现的解决方案

已实现 `IqiyiSigner.kt`，包含：
- ✅ URL签名刷新（更新时间戳）
- ✅ 签名重新生成
- ✅ 参数清理（移除过期参数）
- ✅ 备用URL生成

**但仍然失败的原因：**
- ❌ 缺少真实的设备ID映射
- ❌ 缺少用户认证信息
- ❌ 加密算法可能不完整
- ❌ 爱奇艺可能使用动态密钥

## 可行的解决方案

### 方案1：让发送端（手机APP）负责（推荐）✅
- 手机APP实时获取有效的播放URL
- 立即发送到车机播放
- 减少URL传输延迟

**优点：** 不需要实现复杂的签名算法

### 方案2：使用爱奇艺开放平台API
- 申请爱奇艺开放平台权限
- 使用官方API获取播放URL
- 需要应用审核和认证

**缺点：** 需要企业资质，审核周期长

### 方案3：完全逆向工程签名算法
- 逆向分析爱奇艺APP
- 提取完整的签名算法
- 动态获取密钥

**缺点：** 
- 技术复杂度极高
- 爱奇艺随时可能更新算法
- 可能存在法律风险

### 方案4：使用其他视频平台
- 腾讯视频
- 优酷
- 本地视频文件

**优点：** 实现简单，稳定性好

## 测试建议

由于爱奇艺签名验证的复杂性，建议：

1. **优先测试其他平台**
   - 腾讯视频通常限制较少
   - 优酷相对开放

2. **使用测试视频**
   - 点击车机界面上的"测试视频"按钮
   - 验证播放功能本身是否正常

3. **重新投屏测试**
   - 如果要测试爱奇艺，请立即投屏
   - 减少URL传输延迟

## 技术细节

### 爱奇艺签名参数说明

| 参数 | 说明 | 示例值 |
|------|------|--------|
| tm | 时间戳（毫秒） | 1776125141292 |
| tmi | 时间戳（秒） | 1776125142 |
| tvid | 视频ID | 7245663292806300 |
| k_uid | 用户ID | 6da3377a428d73714a6191d21aa976d9130c |
| sgti | 会话追踪ID | 102_6da3377a428d73714a6191d21aa976d9130c_1776125141292 |
| vf | 验证码 | 7af6e499d8cfa986d70415f341eb43f1 |
| sign | 签名 | 长字符串（MD5或其他加密） |
| px | 加密参数 | base64编码的加密数据 |

### 错误代码说明

| 错误代码 | 含义 | 解决方案 |
|---------|------|----------|
| A00001 | 签名不正确 | 重新投屏获取新URL |
| A00002 | URL已过期 | 重新投屏 |
| A00003 | 设备不匹配 | 使用相同设备或重新获取URL |

## 总结

爱奇艺的签名验证机制非常复杂，完整的实现需要：
- 逆向工程APP
- 动态获取密钥
- 模拟设备指纹
- 用户认证流程

**建议：** 优先使用其他视频平台进行DLNA投屏测试，爱奇艺投屏作为可选项即可。
